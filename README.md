# koa-sequelize-boilerplate REST example

To start and test project follow steps

`git clone git@github.com:ProMasoud/koa-sequelize-boilerplate.git`

`cd koa-sequelize-boilerplate`

`npm install`

`npm start`

`open link http://localhost:3737`

# Features
* Koa 2
* Koa Router
* Koa Cors
* Sequelize v5
* Babel 7 setup
